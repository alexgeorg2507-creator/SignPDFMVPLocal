# Начальные данные SignFinder

Эта папка содержит профили подписантов для первого запуска.
PNG-файлы факсимиле не включены — загрузите их вручную через интерфейс.

## Установка профилей

При первом запуске скопируйте содержимое этой папки в data/api/:

```powershell
Copy-Item -Recurse "setup\signers" "data\api\signers"
```

Затем в SignFinder → Настройки → Подписант загрузите PNG факсимиле для каждого профиля.
